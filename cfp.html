Call for Papers

ICAiC (International Conference on Advances in Computing) invites researchers and practitioners to submit original and high-quality papers that address the following themes, but are not limited to:

Track 1: Intelligent Computing

  • Machine learning and artificial intelligence for data analysis
  • Big data analytics and distributed computing
  • Data mining and knowledge discovery
  • Data visualization and decision support systems
  • Applications of data analytics in various domains

Track 2: Networking

  • Communications
  • Data Networking and Information Security

Track 3: Distributed Computing

  • Sequential, Parallel, High Performance Computing
  • Cloud Computing, Edge Computing, Fog Computing
  • Generative AI (Artificial Intelligence)
  • Embedded Systems(MSE)
  • Internet of Things (IoT)
  • Human-Computer Interaction (HCI) and User Experience
  • Quantum Computing

Track 4: Signal Processing

  • Multimedia
  • Signal Processing

Track 5: Algorithms

  • Modeling, analysis, protocols
